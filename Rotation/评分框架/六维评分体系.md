## 评分维度

| # | 维度 | 分值 | 说明 |
|:-:|:----|:---:|------|
| 📜 | 政策边际变化 | 0-3分 | 打压转默许=2，默许转支持=3，无变化=0，转向打压=-2 |
| 📈 | 价格趋势信号 | 0-3分 | 底部企稳2周=1，连续上涨3周=2，加速上涨=3 |
| 🗣️ | 叙事凝聚度 | 0-3分 | 无人关注=0，小众开始聊=1，机构共识=2，全民热议=3 |
| 💰 | 资金流入迹象 | 0-3分 | 北向/公募(1分) + QFII(1分) + 社保(1分) |
| 🛡️ | 估值安全边际 | 0-2分 | 近3年<20%分位=2，20%-50%=1，>50%=0 |
| 🧠 | **聪明钱共振** | 0-3分 | QFII连续2季加仓(1分) + 社保连续2季加仓(1分) + 两者共同指向(1分) |
| **总分** | | **0-17分** | |

## 评级标准

| 总分 | 评级 | 操作建议 |
|:---:|:----:|---------|
| ≥11分 | ⭐⭐⭐ 核心赛道 | 重点关注但需警惕过热信号 |
| 7-10分 | ⭐⭐ 观察赛道 | 持续跟踪，不深度研究 |
| 4-6分 | ⭐ 备选赛道 | 每月复查一次 |
| <4分 | — | 暂不关注，移除当前视野 |

## 聪明钱信号解读规则

| QFII状态 | 社保状态 | 含义 |
|:--------:|:--------:|------|
| ✅ 加仓 | ✅ 加仓 | **高度确认**，但需警惕是否已过热 |
| ✅ 加仓 | ❌ 未动 | 赛道可能处于中后段，外资独舞 |
| ❌ 未动 | ✅ 加仓 | 赛道早期或偏周期反转型 |
| ❌ 未动 | ❌ 未动 | 极早期或逻辑尚未验证 |
